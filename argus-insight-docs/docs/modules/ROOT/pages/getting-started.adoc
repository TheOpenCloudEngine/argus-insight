= 시작하기
:description: Argus Insight 빠른 시작 가이드

== 사전 요구사항

* Python 3.11 이상
* pip (Python 패키지 관리자)
* Git

== 설치

=== 1. 소스 코드 다운로드

[source,bash]
----
git clone https://github.com/TheOpenCloudEngine/argus-insight.git
cd argus-insight
----

=== 2. Server 설치 및 실행

[source,bash]
----
cd argus-insight-server
make dev      # 개발 의존성 포함 설치
make run      # 서버 실행 (포트 {server-port})
----

서버 실행 후 Swagger UI에서 API를 확인할 수 있습니다: `http://localhost:{server-port}/docs`

=== 3. Agent 설치 및 실행

관리할 대상 서버에서 실행합니다.

[source,bash]
----
cd argus-insight-agent
make dev      # 개발 의존성 포함 설치
make run      # 에이전트 실행 (포트 {agent-port})
----

=== 4. Agent 등록

Server에 Agent를 등록합니다.

[source,bash]
----
curl -X POST http://localhost:{server-port}/api/v1/agent/register \
  -H "Content-Type: application/json" \
  -d '{"host": "192.168.1.100", "port": {agent-port}, "name": "my-server"}'
----

=== 5. 헬스체크 확인

[source,bash]
----
# Server 헬스체크
curl http://localhost:{server-port}/health

# Agent 헬스체크 (Server를 통해)
curl http://localhost:{server-port}/api/v1/agent/{agent-id}/health
----
