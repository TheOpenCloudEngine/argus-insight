= API 레퍼런스
:description: Argus Insight Server API 레퍼런스

== 헬스체크

[cols="1,2,3"]
|===
| Method | Path | 설명

| GET | `/health` | 헬스체크 (status, version 반환)
|===

== 인증 API

[cols="1,2,3"]
|===
| Method | Path | 설명

| POST | `/api/v1/auth/login` | 로그인 (username, password -> access_token)
| GET | `/api/v1/auth/me` | 현재 사용자 정보 조회
|===

=== 로그인 예시

[source,bash]
----
curl -X POST http://localhost:{server-port}/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin"}'
----

== 에이전트 관리 API

[cols="1,2,3"]
|===
| Method | Path | 설명

| POST | `/api/v1/agent/register` | 에이전트 등록
| GET | `/api/v1/agent/list` | 에이전트 목록 조회
| GET | `/api/v1/agent/\{id}` | 에이전트 상세 정보
| PUT | `/api/v1/agent/\{id}` | 에이전트 정보 수정
| DELETE | `/api/v1/agent/\{id}` | 에이전트 등록 해제
| GET | `/api/v1/agent/\{id}/health` | 특정 에이전트 헬스체크
| POST | `/api/v1/agent/health/all` | 전체 에이전트 헬스체크
|===

=== 에이전트 등록 예시

[source,bash]
----
curl -X POST http://localhost:{server-port}/api/v1/agent/register \
  -H "Content-Type: application/json" \
  -d '{
    "host": "192.168.1.100",
    "port": {agent-port},
    "name": "web-server-01",
    "tags": ["web", "production"]
  }'
----

== 프록시 API

에이전트의 API를 Server를 통해 호출합니다.

[cols="1,3,3"]
|===
| Method | Path | 설명

| POST | `/api/v1/proxy/\{id}/command/execute` | 에이전트에 명령 실행
| GET | `/api/v1/proxy/\{id}/monitor/system` | 에이전트 시스템 모니터
| POST | `/api/v1/proxy/\{id}/package/manage` | 에이전트 패키지 관리
| GET | `/api/v1/proxy/\{id}/package/list` | 에이전트 패키지 목록
| * | `/api/v1/proxy/\{id}/\{path}` | 범용 에이전트 API 프록시
|===

=== 원격 명령 실행 예시

[source,bash]
----
curl -X POST http://localhost:{server-port}/api/v1/proxy/{agent-id}/command/execute \
  -H "Content-Type: application/json" \
  -d '{"command": "hostname", "timeout": 10}'
----
