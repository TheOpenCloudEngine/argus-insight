= Argus Insight Agent
:description: Argus Insight Agent 개요

== 개요

Argus Insight Agent는 관리 대상 서버에 설치되어 **root 권한**으로 동작하는 서버 관리 에이전트입니다. Argus Insight Server가 이 에이전트를 통해 원격 서버를 제어합니다.

== 핵심 기능

[cols="1,3"]
|===
| 기능 | 설명

| 커맨드 실행 | 원격 셸 명령 실행, 타임아웃 관리, 위험 명령 차단
| 시스템 모니터링 | CPU, 메모리, 디스크, 네트워크 실시간 수집 및 WebSocket 스트리밍
| 패키지 관리 | dnf/yum/apt 자동 감지, 패키지 설치/삭제/업데이트
| 원격 터미널 | PTY 기반 WebSocket 터미널 세션
| Yum 관리 | yum repo 파일 CRUD, 백업, 패키지 상세 관리
| 시스템 모니터링 (고급) | dmesg, CPU, 네트워크, 프로세스, 디스크 상세 모니터링
| 호스트 관리 | hostname, /etc/hosts, /etc/resolv.conf, ulimit
| 사용자 관리 | 사용자/그룹 CRUD, sudo 권한, SSH 키 관리
| 인증서 관리 | CA 업로드, 호스트 인증서 생성(openssl)
| 파일 관리 | 디렉토리/파일 CRUD, chown/chmod, 압축, 프로그램 실행
| 프로세스 관리 | 프로세스 목록, 시그널 전송, 좀비 프로세스, open files 모니터링
| Prometheus 메트릭 | 호스트 메트릭 수집, Push Gateway 전송
|===

== 기술 스택

[cols="1,2"]
|===
| 항목 | 기술

| 언어 | Python 3.11+
| 웹 프레임워크 | FastAPI + Uvicorn
| 스키마 검증 | Pydantic v2
| 시스템 정보 | psutil
| 설정 | PyYAML (Spring Boot 스타일 변수 치환)
| 메트릭 | prometheus-client
| 기본 포트 | {agent-port}
|===

== 모듈 구조

각 기능 모듈은 `router.py`, `schemas.py`, `service.py` 3파일로 구성됩니다. CLI가 있는 모듈은 `cli.py`를 추가합니다.

[cols="1,1,3"]
|===
| 모듈 | CLI | 설명

| `command` | - | 셸 명령 실행
| `monitor` | - | 시스템 리소스 모니터링 (기본)
| `sysmon` | O | 시스템 모니터링 (고급)
| `package` | - | 패키지 관리
| `terminal` | - | 원격 터미널
| `yum` | O | Yum 리포지토리 및 패키지 관리
| `hostmgr` | O | 호스트 관리
| `usermgr` | O | 사용자 관리
| `certmgr` | O | 인증서 관리
| `filemgr` | O | 파일 관리
| `processmgr` | O | 프로세스 관리
| `metrics` | O | Prometheus 메트릭 수집
|===
