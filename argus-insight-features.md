# Argus Insight 기능 목록

## 프로젝트 구성

| 프로젝트 | 설명 | 기술 스택 | 포트 |
|----------|------|-----------|------|
| **argus-insight-server** | 중앙 관리 서버 | Python, FastAPI, SQLAlchemy 2.0, PostgreSQL/MariaDB | 4500 |
| **argus-insight-agent** | 원격 서버 관리 에이전트 (root 권한) | Python, FastAPI, psutil, prometheus-client | 4501 |
| **argus-insight-ui** | 프론트엔드 웹 UI | Next.js 16, React 19, TypeScript 5.9, shadcn/ui | 3000 |
| **argus-insight-docs** | 기술 문서 | AsciiDoc, Antora 3.x, asciidoctor-pdf | - |

## 기능 분류 체계

| 대분류 | 중분류 | 세부 기능 | 상세내용 |
|--------|--------|-----------|----------|
| **에이전트 관리** | 등록/해제 | 하트비트 기반 자동 등록 | 에이전트가 60초 간격으로 하트비트를 전송하면, 서버가 미등록 에이전트를 자동 등록한다 (UNREGISTERED → REGISTERED). hostname, IP, 시스템 정보, 에이전트 버전을 수집한다. |
| | | 수동 등록/해제 | 관리자가 UI에서 에이전트를 수동으로 등록하거나 해제할 수 있다. |
| | | 에이전트 수정 | 에이전트의 이름, 포트, 태그를 변경한다. |
| | 상태 관리 | 헬스체크 | 개별 에이전트 또는 전체 에이전트에 HTTP `/health` 요청을 보내 응답 여부를 확인한다. |
| | | 연결 끊김 감지 | 백그라운드 태스크가 180초 이상 하트비트 미수신 에이전트를 DISCONNECTED로 자동 전환한다. |
| | | 인메모리 레지스트리 | DB 영속화와 병행하여 인메모리에 에이전트 상태를 캐싱한다. |
| | 분류 | 태그 기반 그룹핑 | 에이전트에 태그를 부여하여 논리적으로 그룹을 분류하고 필터링한다. |
| | 조회 | 에이전트 목록/상세 | 전체 에이전트 목록 조회 및 개별 에이전트 상세 정보(hostname, IP, 버전, 상태, 리소스 메트릭)를 확인한다. |
| **프록시 서비스** | HTTP 프록시 | 커맨드 프록시 | UI의 셸 명령 실행 요청을 대상 에이전트로 전달하고 결과를 반환한다. |
| | | 모니터링 프록시 | 에이전트의 시스템 메트릭(CPU, 메모리, 디스크, 네트워크) 조회 요청을 전달한다. |
| | | 패키지 관리 프록시 | 에이전트의 패키지 설치/삭제/업데이트/목록 요청을 전달한다. |
| | | 범용 프록시 | catch-all 경로(`/{path}`)로 에이전트의 모든 API에 접근할 수 있는 범용 프록시를 제공한다. |
| | WebSocket 프록시 | 터미널 릴레이 | UI ↔ Server ↔ Agent 간 양방향 WebSocket을 중계하여 원격 PTY 터미널 세션을 제공한다. 연결 해제 시 자동 정리한다. |
| **서버 관리** | 서버 등록 | 서버 등록/해제 | 에이전트를 서버(REGISTERED)로 등록하거나 해제한다. 일괄 등록/해제를 지원한다. |
| | | 서버 목록 | 상태, 검색 조건 필터 + 페이지네이션으로 등록된 서버 목록을 조회한다. |
| | 서버 정보 | 호스트 인스펙션 | hostname, IP, 네임서버, 환경변수, 디스크, 리소스, 프로세스, ulimit, sysctl, uname 등 종합 호스트 정보를 조회한다. |
| | | 프로세스 Top | htop 스타일의 실시간 프로세스 뷰를 제공한다. |
| | | 프로세스 목록/Kill | 서버의 실행 중 프로세스를 조회하고 특정 프로세스를 종료한다. |
| | 원격 접속 | 터미널 세션 | hostname 기반으로 WebSocket 터미널을 연결하여 원격 셸 접속을 제공한다. |
| **인증/인가** | 로그인 | 로컬 인증 | SHA-256 해시 비밀번호 검증 후 HMAC-SHA256 서명 토큰을 발급한다. 24시간 만료. |
| | | 토큰 관리 | Base64 인코딩 JSON 페이로드 + HMAC-SHA256 서명 구조의 인증 토큰을 사용한다. |
| | 사용자 정보 | 프로필 조회/수정 | 현재 로그인 사용자의 정보를 조회하고 프로필을 수정한다. |
| **사용자 관리** | 계정 관리 | 사용자 CRUD | 사용자 생성, 조회(목록/상세), 수정, 삭제를 수행한다. |
| | | 중복 확인 | 사용자명/이메일의 유일성을 실시간 검증한다. |
| | | 검색/필터 | 상태(active/inactive), 역할(admin/user), 전문 검색 + 페이지네이션을 지원한다. |
| | 역할/상태 | 역할 변경 | Admin, User 역할을 변경한다. |
| | | 계정 활성화/비활성화 | 사용자 계정을 활성화 또는 비활성화한다. |
| | | 역할 목록 | 시스템에 정의된 전체 역할을 조회한다. |
| **대시보드** | 현황 요약 | 종합 개요 | 등록된 에이전트들의 상태를 집계하여 대시보드 데이터를 제공한다. |
| | | 에이전트 요약 | 에이전트를 online/offline/unknown 상태별로 카운트하여 표시한다. |
| **오브젝트 스토리지** | 파일 작업 | 오브젝트 목록 | S3 ListObjectsV2 호환으로 버킷 내 오브젝트를 조회한다. |
| | | 메타데이터 조회 | HeadObject로 파일의 크기, 타입, 수정일 등 메타데이터를 조회한다. |
| | | 다운로드 | GetObject로 파일을 다운로드한다. |
| | | 업로드 | PutObject로 파일을 업로드한다. 멀티파트 스트리밍을 지원한다. |
| | | 삭제 | 단건(DeleteObject) 또는 일괄(DeleteObjects) 삭제를 수행한다. |
| | | 복사 | CopyObject로 동일 버킷 또는 교차 버킷 파일 복사를 수행한다. |
| | 버킷 관리 | 버킷 목록 | 전체 버킷 목록을 조회한다. |
| | | 사용자 버킷 생성 | 사용자별 전용 버킷을 자동 생성한다. |
| | Presigned URL | 다운로드 URL | 서명된 임시 다운로드 URL을 발급하여 직접 다운로드를 허용한다. |
| | | 업로드 URL | 서명된 임시 업로드 URL을 발급하여 직접 업로드를 허용한다. |
| | 대용량 업로드 | Multipart Initiate | 멀티파트 업로드를 시작하고 Upload ID를 발급한다. |
| | | Part URL 발급 | 각 파트별 업로드 URL을 발급한다. |
| | | Complete/Abort | 멀티파트 업로드를 완료하거나 중단한다. |
| | 고급 기능 | S3 Select | CSV/JSON/Parquet 파일에 SQL 쿼리를 실행하여 필요한 데이터만 추출한다. |
| | | 파일 미리보기 | Parquet, XLSX, DOCX, PPTX 등 다양한 파일 형식의 내용을 미리보기한다. |
| | | 오브젝트 태깅 | 오브젝트에 키-값 태그를 설정/조회/삭제한다. |
| | 설정 | 브라우저 설정 | 파일 브라우저 페이지 크기, 미리보기 최대 행 수 등을 설정한다. |
| **노트/노트북** | 노트북 | 노트북 CRUD | 노트북을 생성, 목록 조회, 수정, 삭제한다. |
| | 섹션 | 섹션 CRUD | 노트북 내 섹션을 생성, 목록 조회, 수정, 삭제한다. |
| | | 섹션 순서 변경 | 섹션의 표시 순서를 재배치한다 (ordered ID 기반). |
| | 페이지 | 페이지 CRUD | 섹션 내 페이지를 생성, 목록 조회, 수정, 삭제한다. |
| | | 페이지 순서 변경 | 페이지의 표시 순서를 재배치한다. |
| | 버전 관리 | 자동 스냅샷 | 페이지 수정 시 이전 내용을 자동으로 버전 스냅샷으로 저장한다. |
| | | 버전 목록/조회 | 페이지의 전체 버전 이력을 조회하고 특정 버전 내용을 확인한다. |
| | | 버전 복원 | 이전 버전으로 페이지 내용을 되돌린다. |
| **보안/인증서** | CA 관리 | CA 인증서 업로드 | CA 인증서 파일을 업로드한다. |
| | | CA 키 업로드 | CA 개인키 파일을 업로드한다. |
| | | CA 상태/조회/삭제 | CA 인증서/키의 상태 확인, 내용 조회, 파일 삭제를 수행한다. |
| | | 자체 서명 CA 생성 | 구성 가능한 파라미터로 자체 서명 CA 인증서를 생성한다. |
| | 호스트 인증서 (Agent) | 호스트 인증서 생성 | CA로 서명된 호스트 인증서를 openssl 기반으로 생성한다 (유효기간 825일, 키 2048bit). |
| | | 호스트 인증서 조회/삭제 | 호스트 인증서 정보 확인, 파일 목록, 파일 삭제를 수행한다. |
| **DNS 존 관리** | 연결 | PowerDNS 헬스체크 | PowerDNS 서버 연결 상태를 확인한다. |

| | 내보내기 | BIND 설정 내보내기 | DNS 존을 BIND 형식의 설정 파일로 변환하여 조회 또는 ZIP 다운로드한다. |

| | 인증 | 쿠키 인증 | 토큰 검증 후 쿠키를 설정하고 Nginx auth-url 엔드포인트로 인증 처리한다. |
| **Unity Catalog 프록시** | 프록시 | 범용 프록시 | Unity Catalog 서버로 모든 API 요청(GET/POST/PUT/DELETE)을 전달한다. Settings에서 UC 서버 URL을 설정한다. |
| **인프라 설정** | 설정 관리 | 전체 설정 조회/변경 | 카테고리별 키-값 설정을 조회하고 변경한다. |

| | | Docker Registry 테스트 | Docker Registry(Zot) 연결을 확인한다. |
| | | Unity Catalog 테스트 | Unity Catalog 서버 연결을 확인한다. |
| | 초기화 | S3 버킷 초기화 | global 버킷이 존재하는지 확인하고 없으면 생성한다. |
| | | UC 메타스토어 초기화 | Unity Catalog 메타스토어를 셋업한다. |
| | 검증 | 파일 경로 확인 | 서버 파일시스템에 특정 경로가 존재하는지 확인한다. |
| **커맨드 실행** | 셸 실행 (Agent) | 비동기 명령 실행 | asyncio.create_subprocess_shell로 비차단 셸 명령을 실행한다. 타임아웃 300초, 출력 1MB 제한. |
| | | 위험 명령 차단 | rm -rf /, mkfs, dd if=/dev/zero, fork bomb 등 위험 명령을 필터링하여 실행을 차단한다. |
| | | 작업 디렉토리 지원 | 명령 실행 시 작업 디렉토리(cwd)를 지정할 수 있다. |
| **시스템 모니터링** | 기본 모니터링 (Agent) | 시스템 스냅샷 | psutil을 이용하여 CPU, 메모리, 디스크, 네트워크를 일괄 수집한다. |
| | | 실시간 스트리밍 | WebSocket으로 5초 간격의 시스템 메트릭을 실시간 스트리밍한다. |
| | 고급 모니터링 (Agent) | 커널 로그 | dmesg 로그를 ISO 타임스탬프 형식으로 조회한다. 로그 레벨 필터링을 지원한다. |
| | | CPU 집계 | top 스타일의 전체 CPU 사용률(user/system/iowait/idle/steal)을 조회한다. |
| | | CPU 코어별 | htop 스타일의 개별 코어별 CPU 사용률을 조회한다. |
| | | 네트워크 트래픽 | 인터페이스별 송수신 바이트/패킷 카운터를 조회한다. |
| | | 네트워크 에러 | 인터페이스별 에러 패킷/드롭 카운터를 조회한다. |
| | | 프로세스 목록 | CPU%, 메모리(VSS/RSS/PSS/USS/SWAP) 포함 프로세스 목록을 정렬 및 페이지네이션으로 조회한다. |
| | | 디스크 파티션 | 파티션 정보와 사용량(전체/사용/가용/사용률)을 조회한다. |
| | | 종합 뷰 (Top) | CPU + 메모리 + 디스크 + 네트워크 + 프로세스 목록을 한번에 조회하는 htop 스타일 종합 뷰를 제공한다. |
| **패키지 관리** | 범용 (Agent) | 패키지 설치/삭제/업데이트 | 시스템의 패키지 매니저(dnf → yum → apt)를 자동 감지하여 통일된 API로 패키지를 관리한다. |
| | | 패키지 목록 | 설치된 패키지 목록을 조회한다. |

| | | 레포 백업 | 전체 .repo 파일을 ZIP 아카이브로 백업한다. |
| | | 패키지 관리 | yum을 이용한 설치/삭제/업그레이드를 수행한다. |
| | | 패키지 조회 | 설치된 RPM 목록, 키워드 검색, 패키지 메타데이터, 소유 파일 목록을 조회한다. |

| | | 세션 관리 | 최대 10개 동시 세션을 지원하며, 120초 유휴 세션을 자동 정리한다. |
| | | 터미널 리사이즈 | JSON 프로토콜(`{"type":"resize","rows":N,"cols":N}`)로 터미널 크기를 동적 조정한다. |
| | | 바이너리/텍스트 메시지 | WebSocket의 바이너리와 텍스트 메시지를 모두 지원한다. |
